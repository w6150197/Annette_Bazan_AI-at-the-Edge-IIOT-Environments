This module combines a case study analysis and a hands-on lab to explore IIoT devices, edge computing, and real-time AI systems. The case study examines the Liverpool Smart Pedestrians project, focusing on edge-based video analytics for smart city traffic monitoring, including system design, YOLOv3-based object detection, and NVIDIA Jetson TX2 deployment. It highlights how edge AI improves real-time decision-making in urban environments.

The lab component involves deploying a simple AI model in a simulated edge environment using Visual Studio Code. It covers the full workflow of loading and preprocessing the MNIST dataset, training a convolutional neural network, converting the model to TensorFlow Lite format, and deploying it using Edge Impulse tools. Together, these activities demonstrate how edge computing enables efficient AI model deployment and real-time inference on resource-constrained devices.
