This module focuses on analyzing Industrial IoT network performance through the concepts of Age of Information (AoI) and Packet Loss Probability (PLP), based on research into reliability trade-offs in IIoT systems. The assignment involves both conceptual understanding and practical machine learning analysis using a simulated IIoT dataset. Key tasks include exploring relationships between network parameters and information freshness, visualizing data trends, and building a Random Forest regression model to predict AoI.

The module also examines how different traffic types and network conditions affect data reliability and latency in IIoT environments. Finally, it explores performance optimization strategies and optional deep learning approaches to further analyze AoI and PLP trade-offs in real-world industrial network applications.
