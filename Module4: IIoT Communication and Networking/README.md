This module focuses on the conceptual design and simulation of an Industrial Internet of Things (IIoT) sensor network. The lab explores how different communication protocols—MQTT, CoAP, and OPC UA—are used to generate, transmit, and visualize real-time sensor data in distributed systems. Using Python and tools such as Visual Studio Code, Mosquitto MQTT broker, and supporting libraries, the project simulates temperature and humidity sensors and demonstrates how data flows across IIoT architectures. The module also includes a reflective journal analyzing learning outcomes, challenges, and the practical applications of IIoT communication protocols in real-world systems.
