This assignment explores the key development environments and tools used in Edge AI and IoT systems. It focuses on understanding how tools such as VS Code, Node.js, Edge Impulse CLI, TensorFlow, Google Colab, and Generative AI coding assistants support the development, training, and deployment of intelligent applications at the edge. The goal is to build a conceptual understanding of how these tools are used in real-world IoT and AI workflows, including data processing, model deployment, and real-time system development.
