This module focuses on AI-driven data analytics for Industrial IoT using time-series forecasting techniques. The lab (L06) involves working with real-world IoT temperature sensor data to perform preprocessing, feature engineering, model selection, and evaluation using forecasting tools such as Nixtla AutoML or Wolfram Language. The analysis includes training predictive models, evaluating performance using metrics like MAE and MSE, and improving results through rolling-origin validation.

The presentation (A06) summarizes the forecasting process, key findings, and recommendations, including the impact of generative models for synthetic data augmentation. Overall, this module demonstrates how machine learning can be applied to IIoT sensor data to enable predictive analytics and improve decision-making in real-time systems.
